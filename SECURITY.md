# Security Policy

AEON Universal Agents is an instruction/documentation repository. It does not contain executable code.

Still, security-sensitive contributions matter because these instructions may influence coding agents.

Please avoid adding guidance that would:

- weaken authentication or authorization
- encourage trusting client-side input
- remove validation
- expose secrets or sensitive logs
- encourage unsafe dependency use
- encourage unverified security claims

If you notice unsafe guidance, open an issue or pull request with a clear explanation.
